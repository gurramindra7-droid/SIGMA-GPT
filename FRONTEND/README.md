# 🚀 SIGMA GPT

A modern AI-powered conversational platform built using React, Node.js, MongoDB, and Groq AI.

SIGMA GPT provides a ChatGPT-like experience with real-time AI responses, authentication, persistent chat history, markdown rendering, code highlighting, and a modern user interface.

---

## 🌟 Features

### 🤖 AI Chat

* Real-time AI responses
* Groq LLM integration
* Markdown support
* Syntax-highlighted code blocks
* Copy response functionality
* Streaming-like chat experience

### 🔐 Authentication

* User Registration

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/sigma-gpt.git

cd sigma-gpt
```

### Frontend Setup

```bash
cd FRONTEND

npm install

npm run dev
```

### Backend Setup

```bash
cd BACKEND

npm install

node server.js
```

---

## 🔑 Environment Variables

Create a `.env` file inside BACKEND:

```env
GROQ_API_KEY=YOUR_GROQ_API_KEY

MONGO_URI=YOUR_MONGODB_URI
```
